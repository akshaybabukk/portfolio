Guardian Key

Developed a password security auditing tool named Guardian Key using Python that checks password strength and verifies if credentials have been compromised through the Have I Been Pwned (HIBP) API. The project utilized SHA‑1 hashing and the k‑Anonymity model to securely query the API without exposing user passwords.
